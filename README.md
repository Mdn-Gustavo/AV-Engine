ANTI VURUS ENGINE IN C++

### About
- this is a small project of mine for my company and for my friends to use

-  This is a simple Anti-Virus Engine that works by scannig your selected directory and identifying and comparing the file with the signatures file ( the file that have all the HASH signatures for the Viruses)
    
### Limitations and Future Updates 

- The ANTI VIRUS ENGINE has some limitations in relation to:

-> Lack of Defenses for Modern Virus
- if hash of the virus is not in the signature.txt file the engine will NOT identify the file as a Virus-

-> Lack of Advance Virus Identification
- The Program has an very basic model of a Virus engine, Good for learning more inside de Architecture, but not as good as a modern anti-virus program.

-> Lack of User-friendlyness
- is not very user friendly and in the ambient i plan to put it i need to think what the user will do with the program.
